## Установка
**Шаг 1**. Вставляем токен своего приложения в `config.py`

**Шаг 2**. Вставляем свой ключ в `config.py` (стандартный ключ: 1111)

**Шаг 3**.
```python
pip install -r requirements.txt
```

**Шаг 4**.
```python
python bot.py
```
